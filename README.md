# Leonida Live

A real-time GTA6 community map — drop and upvote in-game locations, easter eggs, and leaks, with live sync and presence across everyone browsing.

**Live**: https://leonida-live.vercel.app

Built with Next.js (App Router), Supabase (Postgres, Auth, Realtime), and Leaflet. See [`IMPLEMENTATION_PLAN.md`](./IMPLEMENTATION_PLAN.md) for the full build log and architecture decisions.

By [BrewedEntropy](https://github.com/Shravan45).

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000). You'll need a `.env.local` — see `.env.local.example` for the required variables and where to get each one.
